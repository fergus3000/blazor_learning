NGINX 1.19 has a folder /docker-entrypoint.d on the root where place startup scripts executed by the docker-entrypoint.sh script
NB! Be careful with newlines in this sh - CRLF will cause problems, must be LF only!!

docker run -d -p 8081:80 --name wasm-sample -e GOOGLE_CLIENTID=clientid-goes-here blazor-oidp-dockertest
